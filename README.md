## easy-set-zsh-highlighting
该项目是为了在网页上轻松设计出自己满意的`zsh-highlighting`配置文件
右上角的`选择文件`是导入已有的配置
`save`是保存当前调整好的配置

右边可以调整字体`颜色`,`加粗`等选项
左边终端为预览
